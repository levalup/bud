# bud
封装SDL的2D游戏引擎

---

本项目可以使用 `CLion` 打开，也可以使用 `cmake` 直接编译。
依赖有 `SDL2` 、 `SDL2_image` 、 `SDL2_ttf` 和 `SDL2_mixer` 。

---

2016年11月25日初次将项目移到 github 上。

采用 cmake 管理工程，会在当前文件夹`build`目录下编译出`libbud.so`，`main.cpp`只是测试用，并无实际意义。

下一步计划是初步补全文档。
